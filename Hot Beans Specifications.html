    Responsibilities

    As a Web Developer you will be responsible for designing, building and maintaing your sites
    Being able to use scripting languages
    Conferring with teams to solve conflicts and problems
    Directing and performing website updates
    Developing or validating test routines
    Editing existing Website Content
    Maintaining an understanding of the latest Web applications
    Able to back up files from web sites for recovers
    Identifying and correcting problems found from customer feedback
    Evaluating Code to ensure it meets industry standard
    and determining user needs by analysing technical requirements

    Qualifications

    Bachelor's degree in Web Development or a related field (alternatively have relevant experience)
    Solid knowledge and experience in programming applications
    Proficient in JavaScript, HTML and CSS
    Proficient in My SQL
    Dedicated team player
    Ability to thrive in fast-paced environment
    Solid ability in both written and verbal communications
    Knowledge of programming language and technical terminology
    Able to develop ideas and processes and clearly express them
    High degree of independant judgement
    Able to solve complex problems
